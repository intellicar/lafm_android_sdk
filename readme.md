## Installation

### cocoapods install
```
  pod 'LAFM', :git => 'https://github.com/intellicar/ace_ios_sdk.git', :tag => '0.0.81' 
```

### Swift package manager (Recommanded)
```
Xcode File -> add packages -> search for "https://github.com/intellicar/ace_ios_sdk" 
```
